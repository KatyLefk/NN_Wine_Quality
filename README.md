# NN_Wine_Quality
Τίτλος Διπλωματικής Εργασίας: Δημιουργία και Εφαρμογή Τεχνητών Νευρωνικών Δικτύων για την Πρόβλεψη της Ποιότητας Κρασιού με βάση Φυσικοχημικά Χαρακτηριστικά.

Το αρχείο Wine_NN.ipyb δημιουργήθηκε για να τρέχει στο Google Colab. Υπάρχει available for public link: https://colab.research.google.com/drive/1yoYnwjI3WQhX8ZJVUPqVhhs3uKaZ269h?usp=sharing

Τα PUBLIC δεδομένα βρίσκονται στα αρχεία winequality_red.csv και winequality_white.csv και προέρχονται από το 
UCI Machine Learning Repository: https://archive.ics.uci.edu/dataset/186/wine+quality .

Το συμπληρωματικό αρχείο experiments_plan.xlsx περιέχει τα αποτελέσματα των πειραμάτων που έτρεξαν στην παρούσα διπλωματική εργασία και χρησιμεύει στον κώδικα μόνο για να βγουν τα γραφήματα bar charts που δείχνουν την ακρίβεια ανά πειραματικό κύκλο.

Εφαρμογή:
Το μόνο που χρειάζεται για τη χρήση της εφαρμογής είναι επίσκεψη στο σύνδεσμο: 
https://winequalitynn.streamlit.app/ 
Ο κώδικας που χρησιμοποιήθηκε για τη δημιουργία της εφαρμογής είναι:
1. Τα τελευταία 2 κελιά του αρχείου Wine_NN.py
2. Το αρχείο δημιουργίας εφαρμογής wine_qual_app.py
3. Το αποθηκευμένο μοντέλο Νευρωνικού Δικτύου wine_nn_model.h5 (που χρησιμοποιεί το αρχείο 2)
4. Τα αποθηκευμένα αρχεία για Scaler και PCA που αποθηκεύτηκαν από το αρχείο 1 και χρησιμοποιεί το αρχείο 2: scaler.pkl και pca.pkl
5. Το αρχείο requirements.txt που περιέχει όλες τις απαραίτητες βιβλιοθήκες στην έκδοση που μπορούν να συγχρονιστούν.
